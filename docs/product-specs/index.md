# 제품 명세 인덱스

이 디렉터리는 사용자에게 보이는 동작의 공식 기준 문서를 두는 곳이다.

## 현재 범위

- 개인 랜딩 및 홈 화면 경험
- 이력서와 포트폴리오 표현 방식
- Markdown 기반 블로그 발행
- GitHub Pages 배포 파이프라인

## 명세 운영 규칙

- 일시적인 대화가 아니라 안정적인 동작 기준을 이곳에 적는다
- 진행 중인 구현 작업은 관련 실행 계획과 연결한다
- 동작이 바뀌면 같은 PR에서 관련 명세도 함께 갱신한다
